阶段一：核心课程（9 节）—— 广度优先，建立完整体系

覆盖 DevOps 四大支柱：容器化、CI/CD、IaC、监控。学完具备初级 DevOps 工程师的知识广度。

    0001 什么是 DevOps？
    DevOps 解决什么问题、8 阶段无限循环、四大支柱全景。画出你的 DevOps 地图。
    前置：无（第一课）→ 后续：0002
     0002 Linux 命令行基础
    12 个核心命令、管道与重定向、权限模型（rwx/755）、systemd、进程管理。附速查表。
    前置：无 → 后续：0004 / 0010
     0003 Git 进阶
    rebase 交互式、stash、cherry-pick、bisect 二分法找 bug、reflog 后悔药、Git Hooks。
    前置：会基本的 commit/push → 后续：0006
     0004 Shell 脚本编程
    变量、条件判断、循环、函数、set -euo pipefail 错误处理。实战：部署前检查脚本。
    前置：0002（命令行基础）→ 后续：0006 / 0007
     0005 Docker 入门
    Dockerfile → Image → Container。分层缓存、多阶段构建、Docker Compose。实战：容器化 Node.js 应用。
    前置：0002（命令行）→ 后续：0007 / 0014
     0006 CI/CD：GitHub Actions
    Workflow、Job、Step、Runner。Secrets、触发器。实战：给你的项目写 CI 流水线。
    前置：0003（Git）+ 0004（脚本）→ 后续：0007
     0007 部署实战
    SSH 免密登录、Nginx 反向代理、Let's Encrypt HTTPS、Docker Compose 生产配置、部署脚本。
    前置：0005（Docker）+ 0006（CI/CD）→ 后续：0014
     0008 IaC：Terraform
    Provider、Resource、State、Plan/Apply。HCL 语法。实战：用 Terraform 管 Docker 容器。
    前置：0005（Docker）→ 后续：独立模块
     0009 监控 & 可观测性
    Prometheus + Grafana + Node Exporter。PromQL、告警规则、RED/USE 方法。实战：10 分钟搭监控栈。
    前置：0005（Docker）→ 后续：0016

阶段二 · Stage 1：Linux 深入（2 节）

从「会用命令」到「能管服务器」。出问题能自己排查。

    0010 Linux 系统管理进阶
    用户/组管理（useradd/usermod/visudo）、systemd unit 文件编写、cron 定时任务、logrotate 日志轮转、ufw 防火墙、磁盘挂载。
    前置：0002（命令行基础）→ 后续：0011
     0011 Linux 排错实战
    strace、lsof、tcpdump、性能分析套件。5 个经典故障场景：磁盘满、OOM、端口占用、进程死亡、响应慢。
    前置：0010（系统管理）→ 后续：0012

阶段二 · Stage 2：网络深入（2 节）

面试高频题：TCP 握手挥手、DNS 解析链、TLS 证书链、502 vs 504。

    0012 网络协议基础
    TCP/IP 四层模型、三次握手/四次挥手、连接状态（CLOSE_WAIT 告警）、DNS 解析链、TLS 握手/证书链、HTTP 状态码。
    前置：0011 → 后续：0013
     0013 网络实战与负载均衡
    Nginx 生产配置（限流、健康检查、超时）、负载均衡算法（L4/L7）、CDN 原理（缓存穿透/雪崩）、真实排错场景。
    前置：0012 + 0007（部署）→ 后续：0014

阶段二 · Stage 3：Kubernetes（3 节）

容器编排的事实标准。JD 出现率 > 80%，面试必考。

    0014 K8s 核心概念
    架构（Control Plane + Node）、Pod、Deployment（滚动更新/回滚）、Service（ClusterIP/NodePort）、Namespace、kubectl 基础。
    前置：0005（Docker）+ 0007（部署）→ 后续：0015
     0015 K8s 网络与配置
    Ingress（外部流量入口）、ConfigMap/Secret（配置管理）、PVC（持久化存储）、资源限制（requests/limits）、Probes（liveness/readiness/startup）。
    前置：0014 → 后续：0016
     0016 K8s 生产实践
    Helm（包管理器）、HPA（自动扩缩容）、RBAC（权限控制）、K8s 监控集成（Prometheus Stack）、安全清单。
    前置：0015 + 0009（监控）→ 后续：结业项目
