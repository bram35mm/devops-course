Linux 命令速查表

DevOps 日常高频命令。打印出来放桌上，用到熟练为止。

文件操作

ls -la
列出文件（含隐藏，长格式）
cd /path
切换目录
pwd
当前路径
cp src dst
复制
mv src dst
移动/重命名
rm -rf dir
递归强制删除 ⚠️
mkdir -p a/b/c
创建多级目录
touch file
创建空文件 / 更新时间戳
ln -s target link
创建符号链接（软链接）

读文件

cat file
打印整个文件
less file
分页浏览 (q 退出)
head -N file
前 N 行
tail -N file
后 N 行
tail -f file
实时追踪 (Ctrl+C)
wc -l file
行数 (-w 字数, -c 字节)
file file
判断文件类型

搜索

grep pattern file
在文件中搜索模式
grep -r pattern dir/
递归搜索目录
grep -i pattern file
忽略大小写
grep -v pattern file
排除匹配的行
grep -c pattern file
只输出匹配行数
find dir -name "*.log"
按文件名找
find dir -mtime -7
最近 7 天修改的
find dir -size +100M
大于 100MB 的
find dir -exec cmd {} \;
对每个结果执行命令

管道 & 重定向

cmd1 | cmd2
管道：把输出传给下一个
cmd > file
输出重定向到文件（覆盖）
cmd >> file
输出追加到文件
cmd < file
从文件读取输入
cmd 2>/dev/null
丢弃错误输出
cmd &>/dev/null
丢弃所有输出

进程

ps aux
列出所有进程
ps aux | grep name
按名字过滤进程
top / htop
实时进程监控
kill PID
终止进程 (SIGTERM)
kill -9 PID
强制终止 (SIGKILL)
pkill -f pattern
按命令名杀进程
nohup cmd &
后台运行，断线不中断
Ctrl+Z → bg
挂起 → 放到后台

磁盘 & 内存

df -h
磁盘空间概览
du -sh dir/
目录总大小
du -h --max-depth=1
一级子目录大小排行
free -h
内存使用

网络

curl URL
发 HTTP GET
curl -I URL
只看响应头
curl -X POST -d '{}' URL
发 POST JSON
ss -tlnp
查看监听端口
ping -c N host
Ping N 次
dig domain
DNS 查询
nc -zv host port
测试端口是否开放
scp file user@host:/path
远程复制文件

权限

chmod +x script
加执行权限
chmod 755 file
rwxr-xr-x
chmod 644 file
rw-r--r--
chown user:group file
改所有者
sudo cmd
以 root 执行

systemd (服务管理)

systemctl status svc
服务状态
systemctl start/stop svc
启动/停止
systemctl restart svc
重启
systemctl enable svc
开机自启
journalctl -u svc -f
实时日志

杂项（会救命）

man cmd
查看命令手册
history
命令历史
!!
重复上一条（sudo !!）
Ctrl+R
搜索命令历史
Ctrl+C
中断当前命令
Ctrl+D
退出 shell / 发 EOF
alias ll='ls -la'
创建别名
tee file
同时输出到屏幕和文件
jq '.' file.json
格式化 JSON
