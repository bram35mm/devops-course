# 完成 DevOps 核心课程（9 节基础课）

用户完成了从零开始的 9 节 DevOps 核心课程，覆盖了：

1. DevOps 概念 & 全景图（DevOps lifecycle, 四大支柱）
2. Linux 命令行基础（12 个核心命令 + 权限模型 + 速查表）
3. Git 进阶（rebase, stash, cherry-pick, bisect, reflog, hooks）
4. Shell 脚本编程（变量、条件、循环、函数、错误处理、实战脚本）
5. Docker 入门（Dockerfile, Image, Container, 多阶段构建, Compose）
6. CI/CD（GitHub Actions: workflow, job, step, secrets, 触发器）
7. 部署实战（SSH, Nginx 反代, Let's Encrypt, Docker Compose 生产配置）
8. IaC / Terraform（Provider, Resource, State, Plan/Apply, HCL）
9. 监控 & 可观测性（Prometheus, Grafana, Node Exporter, RED/USE, 告警）

**Evidence:** 每节课均有对应的 HTML 文件和测验。用户在所有课程中主动提问并推进下一课。
